简记最近学习2016.3.27

Posted on 2017-03-27 | In Notes | | Visitors

前言

其实并没有什么内容，就是讲讲吐槽一下最近几天学习的一些内容。
前几天搞了Spring Boot的后台，原本想前端用Angular重写的，然而想借这个机会顺便接触点新东西。毕竟现在前端的状态是一觉起来又多了几个框架。

吐槽

看了一下Ant Design，UI的风格个人还是比较喜欢的（相比于Bootstrap），所以界面部分就用它了。既然它本身是React的Component，那前端框架自然也就选用React了。Ant Design的官方项目实战是使用的dva：

dva 是一个基于 react 和 redux 的轻量应用框架，概念来自 elm，支持 side effects、热替换、动态加载、react-native、SSR 等，已在生产环境广泛应用。

说白了是蚂蚁的一套整合了各个框架的Best Practice。那就搞呗。

先走了一遍Dva的Demo Count，没啥大问题，基本只涉及最核心的几个概念，Model、Service、Component。。。。

再看了Demo User Dashboard，感觉不对了。。。

其实这个时候去把阮一峰大神的ES6 教程大致看完了一遍。什么是Generator、yield、async、await。。。。大致了解了一下。

看到文档中用到了Mockjs，就又大概的看了一下它的文档。使用Mockjs可以给前端提供模拟数据，并拦截前端的请求，返回生成的数据。当后台还没准备好数据时，是不错的选择。

回来又继续，发现Redux还没什么概念，现在准备去看文档。

深深感受到了来自了前端的满满恶意…..吐槽完可以继续去滚看文档了。
